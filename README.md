### How to run dev version?

Backend

1. Download repo and `cd into project dir`
2. `python3 -m venv my-venv`
3. `pip install -r requirements.txt`
4. `python manage.py runserver`

Frontend

1. `cd frontend/`
2. `npm i`
3. `npm start`
